## 1、标题
一个#号+空格表示大标题
# 大标题
两个#号+空格表示二级标题
## 二级标题
三个#号+空格表示三级标题
### 三级标题
## 2、引用
一个>+空格
> 能够使该行突出显示的效果
>> 形成嵌套效果（两个>>）
## 3、列表
1. 无序列表
- 列表1
+ 列表2
* 列表3
2. 有序列表
3. 嵌套
    1. 嵌套列表（前空四格）
    2. 上一级列表为1 2    3 ...，嵌套列表为i ii...
4. todolist（格式：- [ ] name   注：中括号里要空格，否则为列表(为了美观，可在-前键入空格)
    - [x] a 
    - [x] b
    - [ ] c
